Welcome to the Rhythm Plus DiscordRPC extension
 

Thank you for installing this extension! This now allows you to show that your playing the Rhythm Plus web game on Discord via rich presence as well as allowing you to show the current song you're playing, the results and current stats!
 

What to do now

If you haven't already, you need to install the Windows app that will receive the stats of your game through the extension and create the DiscordRPC since the extension can't do it itself. You can find it here on the same GitHub repo in the releases section.
 

How to configure

To configure the extension's settings, find the Rhythm Plus icon in the system tray for the 'server' app and press it. You should now see a window appear allowing you to configure the app or right click the icon to get quick options such as closing the server. You can also configure the extension settings by pressing the icon of the extension in the extension menu of your browser.
